# Grabb som springer

Unity-projekt för spelet.

Det här repot är basically bara en dump av Unity-projektets root-folder, så öppna det som ett vanligt Unity-projekt i Unity Hub.

## Starta projektet

1. Clona repot

2. Öppna Unity Hub.

3. Klicka på **Add** / **Add project from disk**.

4. Välj mappen du precis clonade.

5. Öppna projektet i Unity-version **6000.3.15f1**.
